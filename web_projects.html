Web development projects

1. Link People


Job sites are a valuable resource for career and business professionals to network build relationships with potential customers, clients, and partners. LinkPeople is specifically designed for career and business professionals to connect.
This website is based on job sites like Linked-in, Indeed, etcetera, where users can log in, edit his/her profile, search for jobs, access some premium content, and can also post jobs. It's ideal for home-based business owners, freelancers, and telecommuters, as it can help them build their business and career, as well as stay connected to the outside world.


1.1 Technology used

Mainly used Html, CSS, JavaScript, Bootstrap in the front-end, Python and Django in the back-end, and PostgreSQL as the database.

1.2 Features

For Admin:
  • Can log in, log out.
  • Can create new superuser/admin account.
  • Admin can change his profile information.
  • Have the ability to create, update, and delete a user.
  • Can create, update, and delete a user profile.
  • Have the ability to approve, publish, and delete job posts.
  • Can see all applicants and the ability to create & edit/remove them.
  • Also can apply to the job posts.
  • Have access to read, write, and update all premium contents.
  • Can change permission settings of premium contents of any user.
  • Can change site domain name and display name.
  • Can see verified/non verified email addresses.
For User:
  • Can create an account, log in, and log out.
  • Must need to verify the email address.
  • Can reset his/her password.
  • Can edit his/her profile.
  • Can add a profile picture to his/her profile.
  • Can search for jobs by title, skill, company and location.
  • Can apply to the jobs.
  • Have the ability to post a job as a job author.
  • Also can see applicant lists.
  • From the applicant list, the job author can visit applicantsâ€™ profiles.
  • Can access premium content after completing payment process.
  • Can search for blogs in the premium content.

2. Farmer Aider


Every farmer should collect diseased plant samples and take to agriculture specialists to explain plant symptoms. This is a big problem for rural area farmers and new to agriculture. It also takes too much time delay to solve the problem. But this project can give an instant diagnosis to the farmers.
The objective of this project is to provide real-time assistance to farmers who are not getting proper timely treatment for there plants. It is a Q&A based website developed for the assistance of farmers.


2.1 Technology used

Mainly used Html, CSS, JavaScript, Bootstrap in the front-end, Nodejs in the back-end, and MySQL as the database.

2.2 Features

  • There are three types of users, Admin, Farmers(normal user), and Specialists.
  • Farmers and Specialists can create an account for themselves.
  • The farmers can make questions if they need.
  • Admin can approve the questions.
  • There are lots of specialists to give instant responses to the farmer.
  • All of the users can log in.
  • This project can give solutions to various diseases.
  • There is a question bank where all of the farmers can access it.
  • As the queries and answers are stored in the database, all of the farmers can access all of the questions and answers so that they can relate their problems.

Source Code (Github): Farmer Aider

3. Bookstore for Bookworms


It is a Bookstore project where users can find various books, can read reviews of books, can buy them, and get access to all of the books at a time. It is a basic project created for learning purposes. This project is less concentrated in the front-end but enriched in the back-end.


3.1 Technology used

Mainly used Html, CSS, Bootstrap in the front-end, Python, Django in the back-end, and PostgreSQL as the database.

3.2 Features

  • There are two types of users, admin, and normal user.
  • Normal user can sign in and sign up.
  • SMTP mail server is used.
  • Custom User Model and django-allauth is used for authentication.
  • The payment gateway method has created using SSLCOMMERZ so that users can transfer their money securely.
  • Customized admin panel and added filters, so that admin can search objects according to transaction date, card brand, card issuer, etcetera.
  • Even admin can search them in the search bar.
  • Applied unit test for every single part of the project.
  • Custom permission is used for users.
  • Powerful django-debug-toolbar is used to improve performance.
  • Deployment Checklist is used to provide security against SQL injection, XSS attack, Cross-Site Request Forgery, and many more.
  • Finally deployed to Heroku.
