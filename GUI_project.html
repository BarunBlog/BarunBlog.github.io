GUI project

1. Student Admission System.


This is a student admission system project, where a student can enter his/her personal details, background, education, etc. It is a front-end base project, so there is no database in this project.


1.1 Technology used

Mainly used Java and Java Swing to create this project.

1.2 Features

  • There are various types of input fields.
  • Students can enter his\her details.
  • After entering personal information, they will get a confirmation message.
  • Students can get a tuition fee waiver according to their HSC and SSC results.
  • Also, they can see the admission fee structure.
  • After completing the form, they will get a success message.
  • As this project has no database in the back-end, none of the data is stored.

Source Code (Github): Student Admission System.

Note that: .jar file has given in the bin folder, and if you want to run the .jar file, then make sure that all of the images are in the same folder.
