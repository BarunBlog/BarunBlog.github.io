Front-end projects

1. Get Easy Hotel


This is a hotel booking site like Trivago, Booking.com, etc, where users can search according to location or hotel name and book a room. It is a front-end-based project and was created using React.js, so there is no back-end in this project.


1.1 Technology used

Mainly used Html, CSS, JavaScript, and ReactJs to create this project

1.2 Features

  • Users can search hotels by hotel name or location.
  • Can see hotel details.
  • Signup and About us views are created.
  • As this project has no database, all of the data are loading from a variable statically.
  • This project is super responsive.
  • Mainly React function based component used here.
  • React Hook is used to handling states in react.
  • React Router is used for routing.
  • Material-UI is used for user interactivity.
  • Finally deployed to Firebase with GitHub actions.

2. Get Easy Food


Get-Easy-Food is an online food ordering site where users can see different types of food manus and various popular dishes. It is a front-end-based project and was created using Html, CSS, JavaScript, and JQuery, so there is no back-end in this project.


2.1 Technology used

Mainly used Html, CSS, JavaScript, and JQuery to create this project

2.2 Features

  • Users Can find animation effects in each section.
  • Users can see different types of food manus and various popular dishes.
  • In the about-us section, users can see an animated person.
  • This is created using SVG animation with CSS styling.
  • In the menu section, dishes are divided into four sections, Breakfast, Lunch, Dinner, and Dessert.
  • In the popular section, the 3D animation effect is added using Vanilla JavaScript.
  • When the mouse enters the card, the animation effect starts, and the card responds according to the mouse pointer.
  • More dishes are added in the gallery section.
  • An order form is created.
  • Finally deployed to Firebase.

Source Code (Github): Get Easy Food
